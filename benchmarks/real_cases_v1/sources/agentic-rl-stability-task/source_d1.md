## Snapshot digest

RAGEN frames interactive reasoning as trajectory-level multi-turn reinforcement learning in stochastic environments. The project treats agent reasoning as a sequential control problem instead of isolated single-turn prompting, which makes long-horizon stability and credit assignment important.
